# Задание 1

## Задача 1

### Текст задачи

Составить программу, которая находит значение отношения трехзначного
числа к сумме его цифр. 

### Алгоритм решения

Сначала программа просит пользователя ввести трёхзначное число.
Потом она берёт это число и, если оно отрицательное, делает его положительным.
Дальше программа начинает находить сумму цифр с помощью рекурсии. Она берёт последнюю цифру числа с помощью операции mod 10, потом убирает эту цифру с помощью деления на 10 без остатка.
После этого программа вызывает сама себя для оставшейся части числа. Так продолжается, пока число не станет равно нулю.
Когда рекурсия заканчивается, программа начинает складывать цифры и получает сумму всех цифр числа.
После этого она проверяет, не равна ли сумма нулю, чтобы избежать деления на ноль.
Если всё нормально, программа делит исходное число на сумму его цифр и выводит результат.

### Тестирование

Скриншоты результата работы программы 

# Задание 2

## Задача 1

### Текст задачи

Определить количество ненулевых элементов списка. 

### Алгоритм решения

Сначала программа получает список чисел.
Дальше она начинает обрабатывать список с помощью рекурсии, проходя по нему по одному элементу.
Если список пустой, значит элементов больше нет, и программа возвращает 0.
Если первый элемент списка равен нулю, программа его пропускает и продолжает работать с оставшейся частью списка.
Если элемент не равен нулю, программа вызывает сама себя для хвоста списка, а потом добавляет 1 к результату, потому что нашёлся ненулевой элемент.
Так программа проходит весь список, проверяя каждый элемент.
В конце она возвращает общее количество элементов, которые не равны нулю.

### Тестирование

Скриншоты результата работы программы

# Задание 3

## Задача 1

### Текст задачи

Определим множество как список без повторяющихся элементов. Доказать II
дистрибутивный закон. 

### Алгоритм решения

Сначала программа задаёт три множества A, B и C в виде списков.
Дальше она находит объединение множеств B и C, то есть собирает все элементы из этих списков без повторений.
После этого программа находит пересечение множества A с результатом объединения B и C. Это получается левая часть дистрибутивного закона.
Затем программа отдельно находит пересечение множества A с множеством B и пересечение множества A с множеством C.
После этого она объединяет результаты этих двух пересечений. Это получается правая часть выражения.
Дальше программа сортирует обе части, чтобы элементы шли в одинаковом порядке.
После этого программа сравнивает левую и правую части. Если они совпадают, значит дистрибутивный закон выполняется, если нет — не выполняется.
В конце программа выводит обе части и результат проверки.

# Задание 4

## Задача 1

### Текст задачи

Имеется четыре котенка – Дружок, Елисей, Фантик и Мурлыка и четыре
мальчика – Миша, Максим, Леня и Дима. Каждый мальчик взял себе котенка
любимого цвета.
При этом:
– Фантик – не рыжий
– Мурлыка – не серый
– Дружок – не белый
– Елисей – не серый
– У Миши – черный котенок
– У Максима – Мурлыка
– У Лени – Елисей
– У Димы – белый котенок
– Дима не взял Фантика
– Дружок – не серый
– Одно из этих пяти утверждений ложное
У какого мальчика какой котенок?


Сначала программа задаёт список всех котят и список всех цветов.
Дальше она перебирает все возможные варианты распределения котят между мальчиками, то есть пробует, кто какого котёнка мог взять.
После этого программа перебирает все возможные варианты назначения цветов котятам, чтобы у каждого котёнка был свой уникальный цвет.
Затем она связывает котят с их цветами в виде списка, чтобы можно было удобно проверять условия задачи.
Дальше программа проверяет все условия из задачи. Например, что Фантик не рыжий, у Миши чёрный котёнок, Дима не взял Фантика и так далее.
Так как по условию одно утверждение ложное, программа не требует выполнения всех условий, а считает, сколько из них выполняется.

После этого она проверяет, что выполнено ровно 9 условий из 10, то есть одно действительно ложное.
Если это так, программа считает найденный вариант правильным.
Дальше она выводит, какой котёнок у каждого мальчика, а также какого цвета каждый котёнок.
В конце программа останавливается и не ищет другие варианты.

### Тестирование

Скриншоты результата работы программы
